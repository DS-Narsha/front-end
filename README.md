## Installation

### Setup
For building and running the application you need:
- Node.js 16.17.0
- Python 3.10.0
- Npm 8.15.0

### Client
```
$ git clone https://github.com/DS-Narsha/front-end.git
$ cd narsha_front
$ nvm install 16.17.0
$ npm i
$ npm start:dev
```
